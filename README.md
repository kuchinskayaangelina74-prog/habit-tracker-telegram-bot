# 🤖 Habit Tracker Telegram Bot Service

Современная микросервисная экосистема для автоматического трекинга и формирования полезных привычек в течение критического цикла из 21 дня. Проект полностью контейнеризирован и готов к продакшн-развертыванию.

* **Telegram-бот:** [@HabitTrackerCoreBot](https://t.me)

---

## Стек технологий и архитектура

Проект разделен на три независимых изолированных слоя (микросервиса), которые взаимодействуют друг с другом во внутренней сети Docker:

1. **Backend (FastAPI + SQLAlchemy + Alembic):** 
   * Выступает в роли Core API.
   * Обеспечивает безопасность с помощью авторизации по **JWT-токенам** (криптографический хэш PBKDF2-SHA256).
   * Автоматически управляет структурой таблиц базы данных через миграции Alembic при старте.
2. **Frontend (pyTelegramBotAPI + Requests):** 
   * Интерфейс Telegram-бота для взаимодействия с пользователями.
   * Выполняет асинхронные HTTP-запросы к Core API бэкенда.
3. **Scheduler Tasks (APScheduler):** 
   * Фоновый планировщик задач, который запускается каждую полночь.
   * Контролирует правило 21 дня, проверяет прогресс и автоматически архивирует завершенные привычки.
4. **Database (PostgreSQL 15):** 
   * Реляционная база данных для надежного хранения учетных записей, привычек и ежедневных логов выполнения.

---

## 🚀 Быстрый запуск всего проекта через Docker Compose

Благодаря полной контейнеризации, вам не нужно устанавливать Python или PostgreSQL на компьютер локально. Вся система поднимается одной командой.

### 📋 Предварительные требования
Убедитесь, что на вашем компьютере запущен **Docker Desktop**.

### 🎬 Инструкция по запуску:
1. Клонируйте репозиторий с кодом и перейдите в корневую папку:
   ```bash
   cd habit-tracker-telegram-bot
   ```
2. Запустите автоматическую сборку образов и оркестрацию контейнеров:
   ```bash
   docker compose up --build
   ```
3. Дождитесь, пока в консоли побегут цветные логи. Бэкенд автоматически накатит таблицы, а бот выведет строку: `Telegram-бот успешно запущен в режиме polling...`.

---

## 📋 Доступные команды в Telegram-боте
* `/start` — Автоматическая регистрация аккаунта на бэкенде, верификация пароля и получение JWT-токена доступа.
* `/add_habit [название]` — Создание новой привычки в базе данных.
* `/habits` — Просмотр списка ваших активных привычек со счетчиком дней (0/21).
